# 🧩 Noah AI — Mental Wellness Platform

> A free, AI-powered mental wellness companion built for youth.

## 🌐 Live Demo
👉 **[noah-ai-frontend.vercel.app](https://noah-ai-frontend.vercel.app)**

---

## ✨ Features

- 💬 **AI Chatbot** — Empathetic conversations powered by LLaMA 3.3 via Groq
- 📓 **Wellness Journal** — Private journal with mood tracking, saved locally
- 🧠 **Mood Tracker** — Track and visualize emotional patterns over time
- 👥 **Talk to a Mentor** — Book sessions with wellness mentors based on severity
- 🌿 **Mini Chat Widget** — Quick access chatbot from any page
- 📱 **Fully Responsive** — Works on mobile and desktop

## 🚀 Run Locally

1. Clone this repo
2. Clone the backend repo and start the server
3. Open `index.html` in your browser

> Make sure backend is running at `localhost:3000`
